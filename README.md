This project is a Garbage Management System developed using Python and MySQL. The system aims to streamline the process of garbage collection and disposal by allowing users to report garbage locations, schedule pickups, and manage waste efficiently. It includes features like user registration, garbage report submission, collection scheduling, and status tracking, all backed by a MySQL database to securely store data. The project demonstrates practical use of Python for backend logic and MySQL for data management, emphasizing real-world problem-solving in urban sanitation
